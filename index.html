Git Time Metric

Simple, seamless, lightweight time tracking for Git

View on GitHub Download .zip Download .tar.gz

Simple

Install the GTM executable and a plugin for your favorite editor. To enable time tracking for a Git project simply run gtm init within the project's root directory.

Seamless

No need to remember to stop and start timers. GTM knowns when you are working and does it for you.

Lightweight

You will not experience any lag or overhead. The GTM process only runs intermittently to capture the time spent.
